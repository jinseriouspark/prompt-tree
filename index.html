사용 한도 분 휴식 시간 분 체험은 짧게 20초로 보여드려요.

이렇게 동작해요

1
시간을 잰다

고른 사이트를 보고 있는 동안만 시간이 쌓여요.

2
한도를 넘기면 잠근다

사용 한도에 닿는 순간 화면을 나무로 덮어요.

3
나무가 자라면 풀린다

휴식이 끝나면 묘목이 노송이 되고 화면이 열려요.

설치 (크롬, 1분)

  1. 주소창에 chrome://extensions 열고 우측 상단 개발자 모드 켜기
  2. 압축해제된 확장 프로그램을 로드 → 내려받은 extension/ 폴더 선택 (GitHub에서 받기)
  3. 툴바의 🌲 아이콘에서 바이브코딩 / SNS와 한도·휴식 시간을 설정하고 저장

파이어폭스: about:debugging → 이 Firefox → 임시 부가 기능 로드 → manifest.json.

🔒 기록은 전부 내 브라우저 안에만

사용 시간·설정은 storage.local에만 저장되고 어디로도 전송되지 않아요. 네트워크 권한도 없습니다.
