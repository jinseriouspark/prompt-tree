Browser extension

확장 설치

정한 시간을 넘기면 화면을 나무로 덮어 쉬게 합니다. 바이브코딩과 소셜 미디어를 따로 지킵니다. 아직 스토어 미등록이라 개발자 모드로 직접 로드합니다.

GitHub에서 받기

크롬 · 엣지

  1. 저장소를 받습니다: git clone 또는 ZIP 다운로드 후 압축 해제.
  2. 주소창에 chrome://extensions 을 열고 개발자 모드를 켭니다.
  3. 압축해제된 확장 프로그램을 로드 → 받은 폴더 안의 extension/ 선택.
  4. 툴바 아이콘에서 모드와 사용 한도·휴식 시간을 정하고 저장합니다.
  5. 설정을 바꾼 뒤엔 해당 사이트 탭을 새로고침하면 반영됩니다.

파이어폭스

  1. 주소창에 about:debugging → 이 Firefox.
  2. 임시 부가 기능 로드 → extension/manifest.json 선택.

임시 로드는 브라우저를 닫으면 사라집니다. 영구 설치는 서명이 필요합니다.
