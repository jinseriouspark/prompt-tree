나무 문지기 확장 설치

정한 시간을 넘기면 화면을 자라는 나무로 덮어 쉬게 합니다. 바이브코딩(AI)·SNS 두 모드.

GitHub에서 받기 홈으로

아직 스토어 미등록 — 개발자 모드로 직접 로드해요(아래 1분 가이드).

크롬 / 엣지 설치

  1. 저장소를 받는다: git clone 또는 ZIP 다운로드 후 압축 해제
  2. 주소창에 chrome://extensions → 우측 상단 개발자 모드 켜기
  3. 압축해제된 확장 프로그램을 로드 → 받은 폴더 안의 extension/ 선택
  4. 툴바 🌲 아이콘 → 바이브코딩 / SNS와 사용 한도·휴식 시간 설정 후 저장
  5. 설정을 바꾼 뒤엔 해당 사이트 탭을 새로고침하면 즉시 반영

파이어폭스 설치

  1. 주소창에 about:debugging → 이 Firefox
  2. 임시 부가 기능 로드 → extension/manifest.json 선택

임시 로드는 브라우저를 닫으면 사라져요(테스트용). 영구 설치는 서명 필요.
